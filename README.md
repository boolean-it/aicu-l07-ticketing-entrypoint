# Materiali Studenti - Lezione 07

## Cosa Devi Produrre

Per il task:

```txt
Serve creare ticket dal supporto.
```

devi preparare:

- mappa dei punti di intervento (entry-point map);
- file candidati verificati;
- file ammessi e file vietati;
- primo slice;
- prompt patch limitato;
- stop condition;
- reviewer leggero.

## File Da Usare

```txt
template/entry-point-map.md
template/prompt-patch-limitato.md
template/reviewer-leggero.md
```

## Regola

Come visto in aula, riusa il percorso.

Non copiare campi, dati e comportamento del caso discusso.

## Pronto Quando

Sei pronto per il lab quando:

- sai indicare i file letti e perche' contano;
- sai dire cosa non va toccato;
- hai scelto un primo slice piccolo;
- hai un prompt patch che non chiede feature completa;
- hai un reviewer che controlla il diff, non progetta.
